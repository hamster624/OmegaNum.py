# OmegaNum.py
[OmegaNum.js](https://github.com/Naruyoko/OmegaNum.js) ported to Python (depending on hardware the limit is 10{1000}10) <br><br>
## Current functions: <br>
**Operation:** lt, gt, eq, lte, gte, maximum, minimum, tofloat, log, ln, logbase, slog, plog (penta-log), hlog (hexa-log), hyper_log (custom layer log), addlayer, 
add, sub, mul, div, pow, factorial, gamma, floor, ceil, tetration, pentation, arrow, neg
 sqrt, root <br><br>
**Formats:** format, suffix, string, hyper_e <br><br>
**Helpers:** correct, fromstring, fromformat, fromhyper_e
